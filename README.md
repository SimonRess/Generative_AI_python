# Generative_AI_python

## Setup - Project & PyCharm

### Create .venv
PyCharm terminal:
```terminal
# cd <project path>
python -m venv .venv
```

### Activate .venv
PyCharm terminal:
```
.\.venv\Scripts\activate
```

### Install requirements
PyCharm terminal:
``` 
pip install -r requirements.txt
```
